# hr-ai-coach

---

AI HR Coach — это интерактивное веб-приложение на базе GigaChat и Streamlit, которое помогает кандидатам готовиться к собеседованиям. Система сопоставляет ответ пользователя с реальной базой данных мнений десятков HR-директоров.

## Ключевые возможности
* Анализ на основе данных: Сравнение ответа с реальным экспертным контекстом (RAG-подход).
* Кластеризация мнений: ИИ группирует ответы экспертов, показывая разные подходы к одному вопросу.
* Критика и Золотой стандарт: Получение аргументированного фидбека и примера идеально сформулированного ответа.
* Приватность: API-ключ не сохраняется на сервере и используется только в рамках текущей сессии.

## Технологический стек
* Python 3.10+
* Streamlit (Интерфейс)
* GigaChat API (LLM для аналитики)
* Pandas (Обработка экспертных баз данных)

## Деплой на Streamlit Cloud (Инструкция для развертывания)

Вы можете разместить это приложение в облаке, чтобы оно было доступно по ссылке.

### Шаг 1: Подготовка GitHub
Убедитесь, что все файлы (app.py, папка modules, requirements.txt) загружены в публичный репозиторий на GitHub.

### Шаг 2: Создание аккаунта
Перейдите на сайт share.streamlit.io и войдите через свой GitHub-аккаунт.

### Шаг 3: Деплой приложения
1. Нажмите кнопку "Create app", а затем выберите "I already have an app".
2. Выберите ваш репозиторий из списка.
3. Укажите ветку (обычно main) и главный файл приложения — app.py.
4. Нажмите "Deploy!".

### Шаг 4: Настройка и использование
Через 1-2 минуты проект соберется, и вы увидите работающий интерфейс. Теперь вы можете скопировать ссылку из адресной строки. Пользователю достаточно будет:
1. Ввести свой GigaChat API Key в боковой панели.
2. Загрузить файл с базой ответов (CSV/XLSX).
3. Нажать кнопку для начала анализа.

## Структура проекта
* app.py — основной файл интерфейса Streamlit.
* modules/processor.py — логика обработки таблиц с данными.
* modules/llm_engine.py — модуль взаимодействия с GigaChat (аналитика и промпты).
* requirements.txt — список необходимых библиотек.
* background.jpg / hr_anim.json — элементы графического оформления.

---

## Содержимое файла requirements.txt
Для корректной работы приложения в облаке файл должен содержать следующие строки:
```text
streamlit
streamlit-lottie
pandas
gigachat
openpyxl
```
